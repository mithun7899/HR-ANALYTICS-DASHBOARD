# HR-ANALYTICS-DASHBOARD
An HR Analytics dashboard built in Power BI that turns raw employee data into actionable insights on attrition, workforce demographics, tenure, and performance — helping HR teams spot trends and make informed people decisions.

📊 Overview
This dashboard provides HR teams and business leaders with a single-page, at-a-glance view of workforce health — tracking headcount, attrition trends, and the factors (department, salary slab, job role, satisfaction, and age) that correlate with employee turnover.

📈 Visuals Included

Visual	Type	Purpose
Attrition Count by Department	Donut Chart	Shows which departments contribute most to attrition
Attrition by Salary Slab	Clustered Bar Chart	Highlights whether compensation band relates to turnover
Attrition by Job Role & Job Satisfaction	Pivot Table	Cross-tab breakdown for deeper drill-down analysis
Age Group Distribution	Clustered Column Chart	Workforce age demographics
Additional breakdowns	Pie Chart / Area Chart	Supplementary trend and composition views
Filters	Slicer	Lets users filter the report interactively

🚀 How to Use


Download HR-ANALYTICS-DASHBOARD.pbix.
Open it in Power BI Desktop (free).
If prompted, update the data source connection to point to your own HR dataset.
Use the slicer and visuals to filter by department, job role, salary slab, etc.
